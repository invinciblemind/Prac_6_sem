MUD Сервер: Техническая документация

.. title:: Техническая документация MUD сервера

Описание задачи
===============

Разработка многопользовательского игрового сервера (MUD - Multi-User Dungeon)
с поддержкой:

- Асинхронной обработки подключений через `asyncio`
- Игрового поля размером 10x10 клеток
- Перемещения игроков по командам: up, down, left, right
- Создания и перемещения монстров
- Боевой системы с различным оружием (sword, spear, axe)
- Обмена сообщениями между игроками (sayall)
- Автоматического движения монстров каждые 30 секунд
- Мгновенного оповещения всех игроков о событиях

Команды клиента
---------------

- `up`, `down`, `left`, `right` - перемещение по карте
- `addmon <name> hello <text> hp <hp> coords <x> <y>` - создание монстра
- `attack <monster>` - атака монстра (урон 10)
- `attack <monster> with <weapon>` - атака с оружием (sword=10, spear=15, axe=20)
- `sayall <text>` - отправка сообщения всем игрокам
